# DevSphere

A full-stack profile management application built with:

- React (Vite)
- Express.js
- MongoDB
- Mongoose
- bcrypt authentication

## Features
- User Registration
- Login Authentication
- Profile Dashboard
- Edit Profile (MongoDB update)
- Protected Routes

## Run Locally

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm run dev
